# Weather Trend Planner

A component-based React weather dashboard that lets users search for a city, view temperature trends, and get practical forecast insights.

## Current Features

- City search with examples in the input placeholder.
- Current weather snapshot with city, country, temperature, condition, and icon.
- Weekly quick stats: 7-day high, 7-day low, and average rain chance.
- Temperature trend line chart for up to 7 forecast days.
- Insight engine that currently includes:
- Rain alert when rain probability is 50% or higher.
- Very hot day alert when temperature is 34C or higher.
- Very cold day alert when temperature is below 5C.
- Warmest day and coolest day summary.
- Best day to go out based on the lowest temperature point in the graph, with rain chance included in the message.
- Dedicated UI states for loading, API errors, and initial empty view.

## Tech Stack

- React 19 + Vite 8
- Chart.js + react-chartjs-2
- OpenWeatherMap 5-day forecast API

## Local Setup

1. Install dependencies.

```bash
npm install
```

2. Create an environment file from the template.

```powershell
Copy-Item .env.example .env
```

3. Set your OpenWeatherMap API key in .env.

```env
VITE_OPENWEATHER_API_KEY=your_api_key_here
```

4. Start the development server.

```bash
npm run dev
```

5. Build for production.

```bash
npm run build
```

6. Preview the production build locally.

```bash
npm run preview
```

## Available Scripts

## Notes

- If the API key is missing or invalid, the app shows a clear error message.
- Daily forecast values are generated by aggregating 3-hour OpenWeatherMap data points into daily averages.
